CSS Hintergründe

Eigenschaften

Es gibt einige verschiedene Hintergrund Eigenschaften:

background-image
Diese Eigenschaft ist die Location von dem Bild, zum Beispiel h1 { background-image: url('http://website.com/img/background.jpg'); } oder relativer Pfad ohne https: h1 { background-image: url('img/eule.png'); }
background-color
Diese Eigenschaft ist die Farbe vom Hintergrund, zum Beispiel h1 { background-color: pink; } oder alternativ als h1 { background-color: #FFC0CB; }
background-repeat
Diese Eigenschaft beschreibt, wie oft der Hintergrund wiederholt werden soll, zum Beispiel einmalig h1 { background-repeat: no-repeat; } oder auf der Y-Achse h1 { background-repeat: repeat-y; }
background-position
Diese Eigenschaft beschreibt, wie der Hintergrund positioniert werden soll, zum Beispiel h1 { background-position: right; } oder auf der Y-Achse h1 { background-position: bottom; }
background-position: center
Diese Eigenschaft beschreibt, wie der Hintergrund im Zentrum positioniert werden soll, zum Beispiel h1 { background-position: center; }
background: pink url('img/eule.png') center 50% no-repeat;
Diese Eigenschaft beschreibt, wie man alle vorher genannten Eigenschaften in einer Zeile gleichzeitig definiert, zum Beispiel: h1 { background: pink url('img/eule.png') center 50% no-repeat; }
